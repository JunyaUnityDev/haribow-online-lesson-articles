AI（Gemini など）で生成した画像の保存場所。

## ルール

- このディレクトリの画像は **すべて AI 生成**
- 記事埋め込み時のキャプションに必ず「※ 写真はイメージです。AI（Gemini）で生成。」を入れる
- ファイル名規則: `{記事スラッグ}-{章}-{内容を表す短い英単語}.{png|jpg}`

  - 例: `dd-origin-part2-ch2-roxy-club.png`
