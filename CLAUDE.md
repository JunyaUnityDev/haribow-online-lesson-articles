## 記事制作フロー

### ストック管理

USER_BOOK スプシの「記事ストック」シート（`SHEET_NAME.ARTICLE_STOCK`）で全記事のネタ・進捗を管理。

- 列: タイトル候補 / カテゴリ / ステータス / 想定公開日 / スラッグ / ファイルパス / 概要 / メモ / 公開URL
- ステータス: ネタ → 執筆中 → 完成 → 公開済み

### 記事を新規制作する流れ

1. JUNYA がネタを思いつく
2. USER_BOOK の「記事ストック」シートに行追加（タイトル候補・カテゴリ・メモ、ステータス「ネタ」）
3. JUNYA が ChatClaude に「この記事書きたい」と相談
4. ChatClaude が構成案を提示 → JUNYA とミーティング（読者層・結論・構成・文体・参考資料）
5. ChatClaude が `.template/ARTICLE_REQUEST_TEMPLATE.md` を埋めた記事制作指示書を作成
6. JUNYA が Claude Code に指示書を投げる
7. Claude Code が `_drafts/{slug}.md` を作成（ステータス「執筆中」）
8. JUNYA が確認・微調整（ステータス「完成」）
9. 公開時: `_posts/YYYY-MM-DD-{slug}.md` にリネーム移動
   - ステータス「公開済み」、公開URL を記入

### 動画・画像・音源の埋め込み

- 画像: `![alt]({{ '/assets/images/file.jpg' | relative_url }})`
- YouTube: `<div class="video-embed"><iframe src="https://www.youtube.com/embed/VIDEO_ID" ...></iframe></div>`
- note 音源: `<div class="sound-embed"><iframe src="https://note.com/embed/sounds/SOUND_ID" height="275" ...></iframe></div>`
- 各クラスは `assets/css/main.scss` の `.article-body` 内に定義済み

### 対話形式記事（文化 / 歴史カテゴリ）

文化 / 歴史カテゴリの記事は、**対話形式で書く事を許容する**（例外的）。

1. 後述の「キャラクター設定」を参照
2. キャスティング決定（基本: フリッキー × トップ）
3. 記事 md ファイル内で `{% include character.html %}` を使って吹き出しを書く
4. 章タイトルは絵文字 + 章番号（🧠🎒🎛💭🌍など）
5. 強調箇所は `**bold**` で太字
6. 引用・参考文献は記事末尾に「## 参考・出典」セクションでまとめる

```markdown
{% include character.html name="flekky" position="left" text="ヒップホップって何っすか？" %}
{% include character.html name="top" position="right" text="フッ…お前、それも知らねぇのか？" %}
```

position は `left` または `right`。基本は左キャラ=質問役（フリッキー）、右キャラ=教え役（トップ等）。

### 執筆ルール（文化 / 歴史カテゴリ・対話形式）

ChatClaude / Claude Code が文化・歴史カテゴリの対話形式記事（=「キャラが歴史を語るシリーズ」）を書く時の必須ルール。JUNYA から 2026-05-04 確定。

#### 1. 「諸説あります」をどこかに必ず入れる

歴史の記述は研究者によって解釈が分かれる。断定口調を避ける、または以下のいずれかを必ず含める:
- 本文中に「諸説あるが」「一説には」のような留保
- 章末や「参考・出典」セクションに「※歴史の細部には諸説あります」と明示

#### 2. 小学生が読んでもわかる文体

- 難しい漢字・専門用語は避けるか、出てきたら平易な注釈をすぐ後に入れる
- キャラの口調はそのままでOK（例: トップの「フッ…」、フリッキーの「マジで?」）
- 一文を長くしすぎない、改行を多用してリズムを作る

#### 3. 「教科書」ではなく「ドラマ」を書く

人々はドラマを見たい。事実を年号順に並べるだけは絶対にNG。

- 時代背景の描写、登場人物の感情、現場の臨場感を必須で入れる
- 五感（音・匂い・温度・触感・空気）で読者を現場に引き込む
- 例: 「コンクリートの床、チカチカ光る蛍光灯、汗とタバコの匂い」

#### 4. 壮大な文化が生まれた「瞬間」にフォーカス

記事の中心は「その文化 / 技術が生まれた一瞬」。
その瞬間を映画のワンシーンのように丁寧に描写する。

例（HIPHOP起源記事の場合）:
- フォーカス: 1973年8月11日、ブロンクスの団地のレクリエーションルームでハークがブレイクを繋いだ瞬間
- 描写: 場所の匂い、人々の動き、会場の空気が変わる瞬間、最初に踊り出した子どもたち
