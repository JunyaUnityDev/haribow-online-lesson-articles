# HARIBOW Online Lesson - Articles

HARIBOW のオンラインレッスン受講者向け記事サイト。

## URL

https://junyaunitydev.github.io/haribow-online-lesson-articles/

## 構成

- Jekyll + GitHub Pages
- ビルド: GitHub Actions
- 記事: `_posts/` 配下に Markdown で記述

## 記事を追加する

1. `_posts/YYYY-MM-DD-slug.md` を作成
2. Front Matter に `category: mindset` または `category: technique` を指定
3. push すると GitHub Actions が自動でビルド & 公開
