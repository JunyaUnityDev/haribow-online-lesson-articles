## 🔄 第七章: 5要素を強くする「秘密兵器」 — 振り返り（Debrief）

{% include character.html name="flekky" position="left" text="マヤさん、5つ全部分かったっす。でも、どうやって強くするんすか?" %}

{% include character.html name="maya" position="right" text="**振り返り(デブリーフ)**。練習やイベントの後に、みんなで『何が良かったか』『何が改善できるか』を話す時間を作る、それだけ" %}

{% include character.html name="flekky" position="left" text="それだけ?" %}

{% include character.html name="maya" position="right" text="2013年に、Tannenbaum と Cerasoli っていう2人の研究者が、**46の研究、合計2,136人分のデータ** をまとめて分析した。結果、振り返りをするチームは、**平均25% パフォーマンスが上がる**" %}

{% include character.html name="flekky" position="left" text="25%!?やばっ" %}

**「チームがダメだ」と感じた時。**

文句を言う前に、何が足りないかを、知ること。

世界中の研究で分かってる、5つの要素。

1. 心理的安全性（ミスしてもOK、と思える）
2. 信頼性（言ったことを、やる）
3. 構造と明確さ（目標と役割が、見える）
4. 仕事の意味（自分にとって、なぜここか）
5. 仕事の影響（自分たちは、何を変えているか）

そして、強くする秘密兵器は **振り返り（18分）**。

**全部、明日から、自分のチームで始められる。**
