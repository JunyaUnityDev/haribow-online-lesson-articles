ここで、スキンケアの2つの基本も整理しておく。「土台を作る」を、明日からドラッグストアで始められる形に。

### 💧 化粧水 — 肌に水分を入れる

**何ができるか**:
- 洗顔で乾燥した肌に水分を補う
- 後の乳液やメイクのノリが変わる

**選び方**:
- 男性ならドラッグストアの「メンズライン」で OK
- ドラッグストアで 800〜1,500円程度

**使い方**:
1. 洗顔後、タオルで顔を軽く押さえて水気を取る（こすらない）
2. 手のひらに 500円玉サイズを取る
3. 顔全体に押し当てるように馴染ませる（叩かない）
4. 額・鼻・あごの「Tゾーン」も忘れずに

**注意**:
- 顔をタオルでゴシゴシ拭くと肌が荒れる、押さえるだけ
- 1日2回（朝・夜）が目安

### 🧴 乳液 — 水分を閉じ込める

**何ができるか**:
- 化粧水で入れた水分が蒸発しないようにフタをする
- 肌の表面を整えて、メイクの密着度を上げる

**選び方**:
- 化粧水と同じシリーズで揃えると失敗しにくい
- ベタつくのが嫌なら「さっぱりタイプ」

**使い方**:
1. 化粧水の後、肌が湿っているうちに
2. 手のひらに 10円玉サイズ
3. 額・両頬・鼻・あごに置いて、顔の中心から外側へ薄く
4. ベタつきが残ったら、ティッシュで軽く押さえる

**注意**:
- 量が多すぎるとベタつく、少量で薄く
- 朝は薄めに、夜はしっかりめが定番

### 1️⃣ アイライナー — 目元を引き締める

![アイライナーを引く位置]({{ '/assets/images/ai-generated/makeup-intro-misaki-eyeliner.png' | relative_url }})

**何ができるか**:
- 目の輪郭をはっきりさせて、表情を強くする
- 遠くから見ても「目力」が伝わるようになる

**選び方**:
- 初心者は「リキッドタイプ」より「ペンシルタイプ」が失敗しにくい
- 色は「ブラウン」を選ぶ（黒は強すぎる、ブラウンは自然）
- ドラッグストアで 500〜1,500円程度

**使い方**:
1. 鏡を顔の少し下に置く（上から覗き込む形）
2. 目を半分開けて、まつ毛の生え際を「点で繋ぐ」感覚で線を引く
3. 一気に引かない、少しずつ「点を繋ぐ」のがコツ
4. 目尻は 1〜2mm 外側に伸ばすと印象が強くなる
5. 男性なら、引いた後に綿棒で軽くぼかす（線が目立たなくなる）

**注意**:
- 上まぶただけで OK、下まぶたは初心者には難しい
- 失敗したら綿棒 + 乳液で軽く拭けば消せる

### 2️⃣ 下地 — 肌のトーンを整える

![下地を塗る5点 — 額・両頬・鼻・あご]({{ '/assets/images/ai-generated/makeup-intro-misaki-base.png' | relative_url }})

**何ができるか**:
- 肌の色ムラ・赤み・くすみを整えて、舞台でも「顔が飛ばない」状態に
- ファンデーション無しでも肌がキレイに見える

**選び方**:
- 初心者は「BBクリーム」または「色付き下地」
- 自分の肌に近い色を選ぶ（店員さんに聞いても OK）
- ドラッグストアで 800〜2,000円程度
- 男性も普通に買える、店員さんもよく案内してる

**使い方**:
1. 洗顔・スキンケア後、肌が乾いた状態で
2. 指先に「米粒2つ分」を取る（つけすぎ注意）
3. 額・両頬・鼻・あごの5点に置く
4. 顔の中心から外側に向かって、指で薄く伸ばす
5. 髪の生え際・フェイスライン（顔の輪郭）は特に薄く
6. 首との境目で色が変わらないよう、馴染ませる

**注意**:
- 厚塗りすると「メイクしてる感」が出る、薄く広げる
- 鼻の脇のくぼみは指で押し込むように

### 3️⃣ リップ — 顔色を良く見せる

![リップを塗る位置]({{ '/assets/images/ai-generated/makeup-intro-misaki-lip.png' | relative_url }})

**何ができるか**:
- 顔全体の血色感を上げて、生き生きした印象に
- 照明下でも顔が「白く飛ばない」効果

**選び方**:
- 初心者は「色付きリップクリーム」が安全
- 男性入門なら「コーラル系」（オレンジ寄りのピンク、自然）
- 女性なら「赤系」「ベージュピンク」など、衣装に合わせる
- ドラッグストアで 500〜1,500円程度

**使い方**:
1. 唇のカサつきが気になる時は、まずリップクリーム下地
2. リップの先を唇の中央に当てる
3. 中央から外側に向かって、薄くなぞる
4. 男性は「色が乗ったか分からないくらい」が自然
5. 仕上げに、唇を軽く擦り合わせて馴染ませる

**注意**:
- 男性は「ツヤツヤしすぎ」に注意、マットなものを選ぶ
- 食事の前後では塗り直す
