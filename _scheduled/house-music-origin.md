## 🎵 第五章: Frankie の3つの実験 — ハウスの誕生

{% include character.html name="top" position="right" text="Warehouse でのある夜、Frankie はある問題に直面した。<br><br>**『手持ちのレコードが、客の踊り続けたい欲求を満たすには、短すぎる』**<br><br>踊りたい客、足りないレコード。<br><br>そこから、Frankie の実験が始まった" %}

---

{% include character.html name="top" position="right" text="**実験その1: ディスコ冒頭のブレイクの2枚使い**<br><br>同じレコードを2枚買って、**ブレイク(間奏)の部分をループさせる**。Kool Herc がヒップホップで使ったのと同じ技術だ" %}

{% include character.html name="top" position="right" text="**実験その2: オープンリールテープの編集**<br><br>中階に置いたリール・トゥ・リール・テープレコーダーで、**曲の構成を作り直す**。ブレイクを延長、新しい部分を継ぎ足し、独自の長尺バージョンを作る" %}

{% include character.html name="top" position="right" text="**実験その3: TR-808 / 909 のドラム被せ**<br><br>1980年代初頭に発売されたドラムマシン、Roland TR-808 と TR-909。<br><br>Frankie はそれを既存のディスコ曲の上に重ねた。**生のドラムが、電子音と融合した**" %}

{% include character.html name="top" position="right" text="Frankie 自身がこう言ってる。**『Kraftwerk(クラフトワーク、ドイツの電子音楽グループ)が、シカゴでハウスを作る上で主要な構成要素だった』**" %}

---

{% include character.html name="flekky" position="left" text="Kraftwerk?" %}

{% include character.html name="top" position="right" text="ドイツの電子音楽の先駆者だ。Frankie は、80年代初頭の **フィラデルフィア・ソウル** と、**Kraftwerk の電子ビート**、**ヨーロッパのエレクトロニック・ボディ・ミュージック** を、混ぜ合わせた。<br><br>**ディスコの魂 + 電子音 + 倉庫の空気**。<br><br>**全く新しい音** が、ここに生まれた" %}

## 👥 登場人物紹介
