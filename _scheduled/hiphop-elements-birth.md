## ⚔ 第二章: ストリートの戦争、1970年代のブロンクス

{% include character.html name="top" position="right" text="1970年代のブロンクスは、**地獄だった**。<br><br>Cross Bronx Expressway(クロス・ブロンクス・エクスプレスウェイ)っていう高速道路の建設で、街が分断され、住宅価格が暴落。<br><br>仕事を失い、貧困、ドラッグ、放火、廃墟。ニューヨーカーは『Bronx is burning(ブロンクスは燃えてる)』って言ってた" %}

{% include character.html name="top" position="right" text="そんな中、**ストリートを支配していたのは、ギャングだった**" %}

---

{% include character.html name="top" position="right" text="**Black Spades**(ブラック・スペイズ)、**Savage Nomads**、**Seven Immortals**、**Savage Skulls** — 多数のギャングが、街区ごとに縄張りを争ってた。<br><br>抗争、殺人、報復。**若者たちは、ギャングに入るしか選択肢がなかった**" %}

---

{% include character.html name="top" position="right" text="でも、1970年代半ば、**ストリートに変化** が起きる。<br><br>『もう抗争はやめよう』<br><br>そう動き始めた若者たちが、ブロンクスの各地に現れた。Black Spades から離脱した者、Savage Nomads から、Seven Immortals から、Savage Skulls から、それぞれの組織から、**集団的な気付き** が起きていた" %}

{% include character.html name="top" position="right" text="そして1977年11月12日、その流れの中で **Universal Zulu Nation** という組織が結成される。<br><br>名前の由来は、映画『Zulu』(1964年)に描かれた **南アフリカのズールー族の連帯**。<br><br>**ギャングの抗争を、音楽とダンスとアートに置き換えよう** — その動きの旗印になった組織だ" %}

## 🕊 第三章: 暴力から、表現へ — 集団的な転換

{% include character.html name="top" position="right" text="Universal Zulu Nation は、**一人のリーダーの組織じゃなかった**。<br><br>Black Spades から離脱した若者たち、他の組織から離れた者たち、ブロンクスの各街区から集まったメンバー — 多くの若者の参加によって動いた集団的運動だ" %}

{% include character.html name="top" position="right" text="**『もう抗争はやめよう。代わりに、音楽とダンスとアートで競い合おう』**<br><br>この発想は、**ストリート全体の集団的な気付き** だった。一人の天才が思いついた話じゃない。**追い詰められた若者たち全員が、同じ方向を向き始めた瞬間**だ" %}

## 🎤 第五章: エレメンツ② MC — 言葉を撃つ者

{% include character.html name="top" position="right" text="**第二の要素: MC(Master of Ceremony、もしくは Microphone Controller)**<br><br>DJ がレコードを回す間、**マイクを持って観客を盛り上げる人物が必要だった**。<br><br>最初は、**『みんな、手を上げろ!』『乗ってる奴は声出せ!』** みたいな掛け声から始まった。<br><br>でもやがて、**韻を踏んだ言葉、ストーリーテリング** へと進化する。<br><br>これが **Rap(ラップ)** の誕生" %}

---

{% include character.html name="top" position="right" text="1979年、**Sugarhill Gang の『Rapper's Delight』** が、初の商業的にヒットしたラップ曲として発売された。<br><br>でも、その前から、**Cowboy(カウボーイ)、Melle Mel(メリ・メル)、Grandmaster Flash and the Furious Five** など、ブロンクスのストリートで MC たちは活動してた" %}

{% include character.html name="top" position="right" text="MC は、**ストリートの詩人** だった。<br><br>**自分の人生、街のリアル、政治、愛、すべてをマイクで叫んだ**" %}

## 👥 登場人物紹介

{% include character.html name="flekky" position="left" text="オレ、フリッキー!19歳、ダブルダッチ歴1年8ヶ月の大学2年生っす。今回はヒップホップ4大要素の起源を、トップさんに教えてもらう役!" %}

**フリッキー(主人公)**

- 19歳、神奈川出身、大学2年生 + コンビニ夜勤バイト

- カッコつけてるけど、根は純粋

- 口癖:「マジで?」「やばっ」「すげーすげー」
